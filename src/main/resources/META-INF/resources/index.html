RIKO - Stocks Pie Calculator

A Pie is a group of stocks, each with a specific weight within the Pie.

Riko calculates the percentage weight of each stock, so you can easily setup your pie.

Upload a CSV file with a list of stocks in a proper, a resulting CSV file will be generated.

Download the sample file: RIKO-SAMPLE.csv (all three columns are required)

Select a calculation strategy:
