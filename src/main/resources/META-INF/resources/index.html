RIKO - Stocks Pie Calculator

Many investment platforms offers a feature called Pies, where you can group stocks together, making it easy to track and invest in that specific group.

Riko calculates the percentage weight of each stock within the pie, so you can easily setup your pie.

Upload a CSV file with your stocks list to be calculated, a resulting CSV file will be generated.

Download the sample file: RIKO-SAMPLE.csv (all three columns are required)

* Because of numeric rounding precision, the weights are returned with two decimal points.

Select a calculation strategy:

The rating value of each stock will be used for calculation.
The dividend yield value of each stock will be used for calculation.
The ratind and dividend yield values of each stock will be used for calculation. The rating value has 6 out of 10 weight in the calculation formula.
The ratind and dividend yield values of each stock will be used for calculation. The rating value has 7 out of 10 weight in the calculation formula.
